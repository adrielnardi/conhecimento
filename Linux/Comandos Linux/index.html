Comandos de Arquivo

pwd: mostra o caminho

ls: lista conteúdo

ls -l: list - informações detalhadas do conteúdo

ls -la: list all - lista tudo (incluindo temporários e ocultos)

echo: printa

cd: entra no diretório

echo ‘argumento’ > teste.txt : joga o conteúdo e cria um txt chamado teste

cat: lê o arquivo texto no terminal

clear: limpa tela

man: manual que ensina o comando (Ex: man pwd)

whoami: mostra nome do usuário atual

cd..: retorna ao diretório anterior
